# 🎮 Stone Paper Scissors Game

A modern 2-player Stone Paper Scissors game built using **React + Tailwind CSS** with backend API support.

---

## 🚀 Features

- 🎯 2 Player Game
- 🎮 6 Rounds Gameplay
- 🏆 Winner Calculation
- 💾 Save Game to Backend
- 📜 Game History Page
- 🔔 Toast Notifications
- 💎 Modern UI (Glassmorphism + Responsive)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- React Icons
- React Hot Toast

### Backend
- Node.js
- Express.js
- MongoDB
